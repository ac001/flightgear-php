Overview

This presentation uses Slidy, which is a css and javascript slide show engine.

All that is required to create a show is to generate the content in html which are converted into slides when the document loads.

{source lang=html4strict}

Contents

Preparation

Working Directory

Create a directory on your pc eg {$d}

Aquire Template

Download the template.html file and save to {$d}. The template contains links to remote css and javascript so you'll need to be online at the moment.

Cover Slide

Open up the file and find the <--# Slide Show: Start--> comment block. Change the cover slide to your details.

{source lang=html4strict}
