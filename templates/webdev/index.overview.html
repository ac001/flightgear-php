The "design" of this site is intended to be flexible, including more development twoaard internationallisation and automation.

A Zone/Site

A zone/site is considered to be a top level script, such as this "site", webdev.php

The $Site object is used to "control" a site, including navigation, loading classes etc. This is used globally throughout the application.

Navigation, Section and Page

The site structure is designed around a two tier level ie section/page

All urls take the form

webdev.php?section=download
webdev.php?section=download&page=linux

//* TODO is to use mod_rewite for

Templating

The site user $smarty templating http://smarty.net

The smarty templates are in directory templates/*

The web_container.html tempaltes is the "main" template, with the head, body, navigation, images, layout, js, css etc. This is displayed by smarty.

$smarty->display('web_container.html);

The container "includes" the site/section.page.html file ie

{literal}
{* line below includes the content page ie  "templates/site/section.page.html" *}
{include file=$Site->mainPage()};{/literal}

web site "section/pages" html files are in the directory templates/site_id/* in the form

section.html
section.page.html
